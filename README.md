# 🔍 データの照合効率化ツール

2つのリスト（CSV）を突き合わせ、在庫数や金額の不一致や片方にしかないデータを見つけ出すツールです。  
目視でのチェックによる見落としを防ぎ、数時間かかる確認作業の工数を大幅に削減します。

## ✨ このツールでできること

- ✅ 商品IDや注文番号をキーとして、2つのリストを照合します。
- ✅ 一方のリストにのみ存在するデータを特定し、ラベル付けします。
- ✅ 双方に存在するデータの差異（在庫数・金額等）を自動検出し、フラグを立てます。
- ✅ 照合結果を1つのシートにまとめ、加工しやすい形式で保存します。

## 🎯 活用シーン
- システム上の在庫数と棚卸実績の照合
- 注文リストと銀行入金明細の照合
- 異なるシステムから出力した顧客リストの重複・漏れの確認

## 🚀 使い方

### 1. セットアップ（初回のみ）
ターミナルでプロジェクトのフォルダに移動し、以下のコマンドを実行してください。

```bash
pip install -r requirements.txt
```

### 2. 準備
照合したい2つのファイルを data フォルダに入れてください。
※ファイル形式は CSV または Excel に対応しています。

### 3. 実行
以下のコマンドを実行すると、照合を開始します。
```bash
python match_data.py
```

### 4. 結果の確認
output フォルダ内に、照合結果をまとめた 「照合結果.xlsx」 が保存されます。

## ファイル構成
data_matching_tool
├── match_data.py           # 【メイン】データ照合プログラム
├── requirements.txt        # 必要なライブラリ一覧
├── .gitignore              # 不要なファイルの除外設定
├── README.md               # この説明書
├── data/                   # 照合したい元データを入れるフォルダ
└── output/                 # 照合結果の出力先

## 注意事項
・照合するキーの統一: 紐付けの基準となる列名（例：「商品ID」「注文番号」など）を一致させてください。
・文字コード: 文字化けを防ぐため、UTF-8（BOM付き推奨）での保存を推奨します。
